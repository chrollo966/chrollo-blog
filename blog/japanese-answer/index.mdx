# 家庭教師—国語の記述問題の指導
中学受験する生徒に教えた．理科と国語の解答．理科も国語も大体よくできているのだが，国語の記述問題ができない．理解力はあるが基礎的な文章力があまり身についていない状態で，一文の書き始めと書き終わりが対応していないレベル．なぜそうなるのか，生徒の解答作成の様子を眺めると，

1. 問題文を読む
2. 本文中の該当箇所を探す
3. 答えらしき部分を抜き出して解答欄に書き始める
4. 再び本文に戻る
5. 再び答えらしき部分を抜き出して解答欄に書く

これを繰り返す．つまり，一文の構成をほとんど意識せず，解答欄と本文の間を行き来しながら場当たり的に解答を書いてしまうので，はじめと終わりの一致しないような，破綻した文を書いてしまうのだ．

これを避けるためには，**解答を書き始める前**に，解答に必要な要素を**箇条書き**にして，それを自然な文章に組み直すためにはどういう配列にすればよいか考えてから解答を書き始めるべきだ．

今後，国語の記述問題を指導するときには以上述べたことを丁寧に説明する．
