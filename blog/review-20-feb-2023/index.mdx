# 自分の飲酒状態を整理する．

酒をやめたいので，記録をつけることにする．

まず酒を飲んでいるとダメなこと：

- 寝るのが遅くなる
- 起きるのも遅くなる
- 酒を飲んでいる間ほとんど集中できないし記憶力も低下する
- 二日酔いで日中何も作業できない

すなわち限りある人生を無駄にしている．

酒をやめてうれしいこと：

- 二日酔いじゃないので日中の時間を使える
- 早寝早起きになるうえにアルコールを摂取しないので健康によい

# 飲酒理由

酒を飲む理由：習慣化してしまい飲まないと眠れない気がするだけ

酒を飲んでしまうトリガー：仕事から帰って椅子に座ってしまったとき

# 飲酒を避けるには

酒を飲まないためには，そのトリガーを避けよう．

トリガーを避けるには：

- 夜は何もしないでベッドに横になる
- 眠れないならベッドで本を読むなどする
- 眠れなければ朝まで起きていてもよい

これを継続するためには：日記をつけて記録するようにする
